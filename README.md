# intelligent-customer-service-agent-simulator

模拟智能客服多轮对话与意图分析

项目经历：我曾参与“智能客服Agent优化”项目，负责协助产品经理进行对话流程设计与评测。我们基于大模型构建了多轮对话能力，并通过分析用户query数据优化了意图识别模块。我协同算法团队清洗了数千条对话数据，重新标注了关键场景，使Agent的意图准确率提升了15%。项目后期，我独立输出了包含bad case分析和改进建议的评估报告，为后续迭代提供了明确方向。

## 运行
```bash
pip install -r requirements.txt
python main.py
```
